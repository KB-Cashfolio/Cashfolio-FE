# 💸 CashFolio

> 감성 + 기록 + 재미를 더한 **가게부 플랫폼**

CasHFolio는 단순한 숫자 기록을 넘어서
키우기 게임과 같은 재미 요소를 결합환 함께 소비를 기록하고,
마치 "거지 키우기"처럼 나의 소비 습관을 시각적으로 확인할 수 있는 가계부입니다.

## 핵심 컨셉

> “가계부를 재미있게 만들 수 없을까?”

CasHFolio는 단순한 기록 도구가 아니라
